# Решатель квадратных уравнений

Функция `get_roots` находит корни квадратного уравнения.

# Как использовать

Пример запуска:

```python
from quadratic_equation import get_roots

print(get_roots(1, 1, -2))
```

Результат:

```python
(-2.0, 1.0)
```
